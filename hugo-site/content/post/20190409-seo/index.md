# Hugoで作ったサイトをGoogleに拾ってもらう

[Google Search Console](https://search.google.com/search-console/about) にドメインを登録しよう

次にサイトマップを登録する。

Hugoの場合サイトマップは /index.xml に生成されている。ので、これをGoogle Search Consoleに登録しよう。反映には時間がかかるようだ。
